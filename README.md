# Titanic Survival Prediction

Проект по машинному обучению, который предсказывает выживание пассажиров Титаника с использованием Логистической Регрессии.

## Описание

В данном проекте используется набор данных Titanic. Модель обучается на различных признаках пассажиров (класс каюты, пол, возраст, цена билета и другие) для бинарной классификации: выжил пассажир или нет.

Основные этапы работы:
1.  **Предобработка данных:** Заполнение пропусков (возраст, цена), кодирование категориальных признаков.
2.  **Feature Engineering:** Создание категорий для цены билета (Fare Categorization) для улучшения корреляции.
3.  **Масштабирование:** Использование StandardScaler для нормализации данных.
4.  **Модель:** Логистическая Регрессия (Logistic Regression).

## Результаты

Точность модели (Accuracy) на тестовых данных: **~79%**.

Анализ весов модели показал, что наиболее значимыми факторами для выживания являются пол пассажира и класс каюты.
